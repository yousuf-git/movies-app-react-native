*   **Discover Movies**: Browse a vast collection of movies.

## üöÄ Installation

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Node.js (LTS version recommended)
*   npm or Yarn
*   React Native CLI or Expo CLI (depending on your project setup)

### Steps

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yousuf-git/movies-app-react-native.git
    cd movies-app-react-native
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Link native dependencies (if applicable):**
    ```bash
    npx react-native link
    ```

## üèÉ Usage

### Running on iOS

1.  **Install CocoaPods (if not already installed):**
    ```bash
    cd ios
    pod install
    cd ..
    ```
2.  **Run the app:**
    ```bash
    npx react-native run-ios
    ```
    Alternatively, open `ios/your-movie-app.xcworkspace` in Xcode and run from there.

### Running on Android

1.  **Start the Metro bundler:**
    ```bash
    npx react-native start
    ```
2.  **Run the app on an emulator or device:**
    ```bash
    npx react-native run-android
    ```
